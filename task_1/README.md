# Task 1 - GitHub, Virtual Environment and Linux Basics

## Description

This folder contains my solution for Task 1 of the Synergy Task Phase. I learned how to create a virtual environment, use Linux commands, and make a project using Git and GitHub.

## Setup Instructions

1. Open a terminal and navigate to the `task_1` folder.
2. Create a Python virtual environment.
3. Activate the virtual environment.
4. Install the required packages.
5. Run the Python program.

## Creating and Activating the Virtual Environment

```bash
python -m venv myenv
source myenv/Scripts/activate
```

## Install the Required Packages

```bash
pip install -r requirements.txt
```

## Run the Python Program

```bash
python task_1/src/hello.py
```
