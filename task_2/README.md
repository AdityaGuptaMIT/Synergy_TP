# Task 2 - Student Submission Analyzer

## Description

This folder contains my solution for Task 2 of the Synergy Task Phase. In this task, I used student submission data and made a summary for it in JSON format as output using Pandas library.

## Setup Instructions

1. Open a terminal and navigate to the `task_2` folder.
2. Create a Python virtual environment.
3. Activate the virtual environment.
4. Install the required packages.
5. Run the Python program.

## Creating and Activating the Virtual Environment

```bash
python -m venv myenv
source myenv/Scripts/activate
```

## Install the Required Packages

```bash
pip install -r requirements.txt
```

## Run the Python Program

```bash
python task_2/src/main.py
```

## Output

The program creates the following file after analysing the data :

```text
task_2/output/summary.json
```
